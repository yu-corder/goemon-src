# Goemon (五右衛門)

Goemon is a stack-machine-based programming language, compiler, and virtual machine implemented in C.

The project was created to explore compiler construction, virtual machine design, parsing techniques, and low-level system architecture through a hands-on approach.

## 🌟 Features

### Language Features

* Integer variables and assignment
* Arithmetic expressions

  * `+`
  * `-`
  * `*`
  * `/`
  * `%`
* Comparison operators

  * `<`
  * `<=`
  * `>`
  * `>=`
  * `==`
  * `!=`
* Conditional branching

  * `if`
  * `else`
  * `else if`
* Loop statements

  * `while`
* Increment operator

  * `++`
* Nested control flow

### Compiler Features

* Recursive Descent Parser
* Operator precedence parsing
* Two-pass compilation
* Backpatching for jump resolution
* Bytecode generation

### Virtual Machine Features

* Stack-based execution model
* Variable storage and memory management
* Conditional jumps (`OP_JZ`)
* Unconditional jumps (`OP_JMP`)
* Arithmetic instructions
* Comparison instructions

---

## 🛠️ Technical Stack

* Language: C
* Architecture: Stack Machine
* Parsing Strategy: Recursive Descent Parsing
* Compilation Strategy: Two-pass Compilation
* Execution Model: Bytecode Virtual Machine

## 🚀 Example Programs

### Conditional Branching

```goemon
a = 10;
b = 20;

if (a < b) {
    print 100;
}
else {
    print 200;
}
```

Output:

```text
VM Output: 100
```

---

### While Loop

```goemon
c = 1;

while (c < 10) {
    print c;
    c++;
}

print c;
```

Output:

```text
VM Output: 1
VM Output: 2
VM Output: 3
VM Output: 4
VM Output: 5
VM Output: 6
VM Output: 7
VM Output: 8
VM Output: 9
VM Output: 10
```

---

### Fibonacci Sequence

```goemon
a = 0;
b = 1;
i = 0;

while (i < 10) {
    print a;

    temp = a + b;
    a = b;
    b = temp;

    i++;
}
```

Output:

```text
VM Output: 0
VM Output: 1
VM Output: 1
VM Output: 2
VM Output: 3
VM Output: 5
VM Output: 8
VM Output: 13
VM Output: 21
VM Output: 34
```

---

### Factorial

```goemon
i = 1;
fact = 1;

while (i <= 5) {
    fact = fact * i;
    i++;
}

print fact;
```

Output:

```text
VM Output: 120
```

## 🔧 Build

Compile the compiler and virtual machine:

```bash
make
```

Compile a Goemon source file:

```bash
./kama-c examples/study.goe
```

Execute generated bytecode:

```bash
./kama-e examples/study.gb
```
